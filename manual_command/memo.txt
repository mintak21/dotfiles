コマンドではやりにくい初期設定内容

0. brewとgitをインストール
> xcode-select --install (brew bundleがうまくいかないものがあるため)

※ここはgitが入っていない場合のみ実行
> /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
> brew install git

1-0. Githubアカウント設定に必要な手順
ここは、使用するアカウントを確認して設定する。

・sshキー(https接続であれば不要)
> sudo mkdir ~/.ssh
> sudo chmod 755 ~/.ssh
> ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
*passphraseのところで、mintak21を入力
> pbcopy < ~/.ssh/id_rsa.pub

最初のcloneのための設定、※global設定は後のdotfilesで上書きされてしまうので要注意
>git config --global user.name "First-name Family-name"
>git config --global user.email "username@example.com"



1-1. ドットファイルのシンボリックリンクを通す
システム環境設定->セキュリティとプライバシーで鍵をあけておく。※こうしないとbundleで失敗することがある。
> cd ~
> git clone git@github.com:mintak21/dotfiles.git
> sh setup_dotfiles.sh
※しばらく時間がかかる。ときどきpassword入力を求められる。

codeコマンドを通す。(VSCode起動->F1->shellでcodeをPATHに通す)
> sh setup_vscode.sh

1-2. go/Rubyをインストール
> goenv install {VERSION}
> goenv global {VERSION}
> goenv rehash
※rbenvでも同じことを行う

1-3. VSCode/Goの設定
 Terminalからcodeで起動
 設定でformatToolをgoimportsにしておく(setting.jsonでなっているはずではあるが)
 F1->Go Install/Upgrade ToolですべてにチェックをいれてOK


2. Chrome拡張機能をいれる
　extensions.htmlのリンクを１つずつたどってインストール
　vimiumはmappingとsearchエンジン設定を同一ディレクトリのテキストをコピペ


3. キーボード
　システム環境設定->キーボード
　・ファンクションキーにチェック
　・ユーザ辞書に名前登録
　・DockとLaunchPadのショートカットを設定
　・Mission Controlでデスクトップ表示のSC(F11)を解除

4. アイテムショートカット登録
　・Cmd+SpaceでAutomatorで検索して開く
　・ワークフロー->アプリケーションを起動->Chrome/Terminalを選んで保存
　・システム環境設定より、キーボード->サービス->一般でショートカット登録

5. ClipyとShiftitのショートカット設定

6. Dockからいらないものは消す
